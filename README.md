# BIOE591: Genomics for Ecology and Conservation

This is the GitHub Repository for the BIOE591 at Montana State University. 

## Scripts

- `hello.py`: Multiprocessor `Python` script for Slurm demo

- `slurm_demo.sbatch`: Slurm job scheduler script  

## Tutorials

- `01_setup.md`: Instructions for setting up your GitHub account and Tempest working directory.

- `02_slurm_demo.md`: Instructions for running the [Slurm Workload Manager](https://slurm.schedmd.com/documentation.html) on the cluster.
